# Svellheim Entities

NPCs, monsters, items, projects, imbuing projects, and rewards for the Svellheim campaign setting (Draw Steel / Foundry VTT).

## Structure

- `data/` - Source JSON entity files
  - `monsters/` - Monster actor definitions
  - `npcs/` - NPC actor definitions
  - `items/` - In-game item definitions
  - `projects/` - Downtime project definitions (Crafting, Research, Other)
  - `imbuing-projects/` - Imbuing project definitions (Armor, Implement, Weapon)
- `module/packs/` - Compiled Foundry VTT LevelDB packs
- `tools/` - Build scripts for compiling packs
